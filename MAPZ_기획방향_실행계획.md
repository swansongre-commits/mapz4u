## 1. 서비스 정의

**한 줄 정의**: 전국의 어린이 체험활동 정보를 상시 수집해, **주제(아이의 언어) × 조건(부모의 언어)**으로 구조화한 지도형 웹서비스. 관심이 있으면 **검색**하고(깊게), 없으면 **둘러본다**(넓게). 웹은 정보 계층이고, 활동은 현실(박물관·체험처·도서관)에서 일어난다.

### 1.1 네이밍

- 영문 **MAPZ** (= MAPS Kids), 한글 표기 **'맵즈'로 고정**, 부제 상시 병기: **"맵즈 — 어린이 체험지도"**
- 정합성: MAPS(기존 백엔드 자산) → MAPZ(어린이 대문) → 제품 실체가 문자 그대로 '지도'. 이름이 아키텍처(Layer A = MAPS 임베딩)와 사용자 성장 경로(MAPZ 어린이 → MAPS 진로/고교)를 인코딩
- 리스크: 한국어 발음상 MAPS와 동음(맵스/맵즈) → 한글 표기·부제 병기로 대응, 패밀리 브랜드로 활용
- [ ] **확인 필요**: 상표 선등록 여부(제9·41·42류), 도메인 가용성

### 1.2 비목표 (Non-goals)

적성 검사·판정 ❌ / 직업 매칭·추천 ❌ / 콘텐츠 시청 서비스 ❌ / 학습 성취 향상 ❌ / 체류시간 극대화 ❌ / **독서 교육·독서 기록 서비스 ❌ (도서는 지도의 부속)**
— 화면의 임무는 최단 시간에 아이를 현실로 내보내는 것.

## 2. 확정 원칙 (제1원칙 토론 산출)

### 2.1 합의 사실 (통합 Fact Base)

**1차 — 어린이 진로 일반 (F)**
- F1. 만 12세 이전 흥미는 불안정. 저연령 적성·흥미 검사의 장기 예측타당도 근거 없음
- F2. 아동은 선택지를 넓히는 게 아니라 고정관념(성역할 6~8세 → 사회적 지위 9~13세, Gottfredson)으로 좁힘. 정당한 개입 목표는 **조기 배제의 지연**
- F3. 미래 직업 지형은 특정 불가 → 매칭은 원리적으로 무의미. 직업명은 사라져도 **동사**는 남는다
- F4. 고교학점제(2025~)로 실질적 첫 의사결정은 고1 선택과목. 그 입력값은 이전에 축적된 자기이해
- F5. 구매 결정자는 부모, 구매 동기의 상당 부분은 불안
- F6. 아이의 지속 사용 조건은 내적 동기(자율성·유능감·관계성, SDT)
- F7. 종단 기록은 자산이자 위험(라벨링·자기충족예언·개인정보)

**2차 — 흥미의 작동 방식 (G)**
- G1. 흥미는 발견이 아니라 **발달**이다 (촉발 → 유지 → 개인적 흥미, Hidi & Renninger)
- G2. 촉발 연료는 놀라움·자기관련성, 유지 연료는 이해·유능감
- G3. **재방문 없는 노출은 흥미를 만들지 않는다** (재방문 대상은 화면이 아니라 주제)
- G4. 전이의 결정 변수는 어른의 반응. 성과화하는 순간 흥미는 죽는다
- G5. 체류시간 ≠ 흥미. 흥미의 지표는 **화면 밖 연장 행동**(질문·놀이·"또 가자")

**3차 — 체험지도의 근본 문제 (H)**
- H1. 체험 공급은 존재하나 파편화. 근본 문제는 공급이 아니라 **발견 비용** (탐색 폭 축소의 실제 메커니즘)
- H2. 체험 정보는 부패한다. **신선도가 신뢰의 전부**, 수집 파이프라인이 서비스의 본체
- H3. 아이는 주제(공룡)로 찾고, 부모는 조건(연령·거리·비용·예약)으로 거른다 → **이중 분류 필수**
- H4. 카테고리 전체의 동시 진열이 곧 탐색 폭 확장 장치 (개입이 아니라 진열 — '열린 서가')
- H5. 성공 지표는 체류가 아니라 **'나가서 다녀옴'**

### 2.2 제품 헌장

1조. 배제를 늦춘다 (결정을 돕지 않는다)
2조. 판정하지 않는다 (등급·백분위·또래비교·유형 라벨 금지)
3조. 불안으로 팔지 않는다 (설계 근거와 마케팅 문구의 분리)
4조. 화면은 예고편이다 (현실 전환이 목적)
5조. 기록은 아이의 것이다 (삭제권, 소급 열람 기본 차단)

### 2.3 폐기·금지 목록

저연령 적성검사 / 직업 추천 알고리즘 / 등급형 리포트 / 또래 비교 / "지금 안 하면 늦는다"류 카피 / 자동재생 / 인기순 단일 정렬 / 롤모델 지정 기능 / 흥미 단계의 외부 노출 / 체류시간 KPI / **필독·권장·수준별 등 독서교육 언어 / 독서 레벨 지수 노출**

## 3. 정보 구조 — 3축과 우선순위

**대문 = 관심사(주제) / 통로 = 인물 / 골조 = 하는일(동사)**

| 축 | 역할 | 근거 |
|---|---|---|
| 관심사(주제) | 대문 (촉발) | 사전지식 제로 + 자기관련성 최대. 고정관념 하중 최소. 실제 검색 행동(H3)과 일치 |
| 인물 | 통로 (유지) | 주제관 내부의 이야기 콘텐츠. 현장→인물 방향. 콘텐츠 단가·권리 리스크 최소 |
| 하는일(동사) | 골조 (전이) | 백엔드 태그로 전체 연결. 직업명이 사라져도 유효(F3) |

- 이 순서는 흥미 발달 경로(촉발→유지→전이)와 동형
- **연령별 표면화 규칙**: 초등 = 동사·사람 / 중등 = 직업명 등장 / 고등 = 학과 (연령이 올라갈수록 명사를 늘린다)
- 관심 없는 아이의 대문: 글자 목록이 아니라 **그림 타일 격자** + 발견 피드의 인물 이야기를 촉발 훅으로
- **도서의 지위**: 제4의 축이 아니라 모든 축에 부속되는 **상시 재고**다. 부패하지 않고(H2 신선도 부담 0), 전국 균등하며(도서관 인프라 = 지역 격차의 최종 해법), 재방문(G3)의 최저비용 수단이고, 체험의 전후를 잇는 다리다("가기 전에 볼 책 / 다녀와서 볼 책")

## 4. 데이터 아키텍처

### 4.1 그래프 노드 (5종)

`주제(Topic)` — `체험항목(Experience)` — `인물(Person)` — `직업·학과(Job/Major)` — `도서(Book)`

### 4.2 엣지

- **동사 태그**: 만들다·돌보다·탐구하다·설득하다·표현하다·조직하다·움직이다 등 7±2개 (O*NET Work Activities를 어린이용으로 번안)
- **의미 엣지(인접성)**: BGE-M3 임베딩 유사도 (MAPS 자산 재활용) — '옆으로 한 칸' 확산 추천의 근거. 수렴형 "이어보기" 금지, 확산형(인접 이동)만
- **행동 엣지(함께 빌린 책)**: 정보나루 recommandList(마니아·다독자 추천)의 대출 동시발생 데이터 = 아이들의 실제 행동적 인접 신호(공룡→화산). 의미 엣지와 **RRF 융합**(MAPS 하이브리드 검색 패턴 재활용)해 인접 그래프 품질 향상. **화면 노출 순위로 직결 금지** — 엣지·후보 풀로만 사용(인기 집중 재생산·부모 매개 편향 통제), 노출은 편성 쿼터가 결정
- **인물–직업 계보**: one-to-many. 예) 김정호(지도 만드는 사람) → 측량사·GIS전문가·내비게이션 개발자·지도 디자이너 / 허준 → 의사·한의사·약사·의학연구자. 매칭의 정반대 방향(하나가 여럿으로 퍼짐)
- **인물–연고지**: 생가·기념관·유적 → 체험항목과 연결, '인물 코스' 주말 카드 포맷

### 4.3 학과·직업 데이터의 3층 부착 (MAPS 자산 활용 설계)

| 층 | 내용 | 노출 |
|---|---|---|
| **Layer A** | MAPS 임베딩(직업 549·학과 504, FAISS)을 그래프 엣지로만 사용. 추천 품질 향상 | 화면 비노출 (전연령, 리스크 0) |
| **Layer B** | 주제관 '이 세계의 어른들' — 인물 이야기 형태. 방향은 **체험→직업 한 방향만**(확장 O, 매칭 X) | 초등, 선택적 |
| **Layer C** | 정식 학과·직업 정보. 자유학기 → 고1 과목 내비게이션 → 기존 고교 진학 서비스 접속 | 중등·부모 화면 |

**Layer B 변환 규칙**: 유지(하는 일·하루·관련 주제) / 제거(연봉·전망·요구 학력 = 지위 신호) / 변환(직업명 → 동사 서술)

### 4.4 체험 항목 스키마

명칭 / 유형(상설시설·기획전시·프로그램·행사) / **주제 태그**(관리형 사전, 동의어 통합: 공룡=화석=쥬라기) / 동사 태그(선택) / 위치·좌표 / 기간·회차·운영시간 / 권장 연령(참고 표기 — 수준 등급처럼 보이지 않게) / 소요 시간 / 비용 / 예약 여부+링크 / 실내외 / 주차·유모차·수유실 / 혼잡 시간대 / **관련 도서(related_books: 가기 전 / 다녀와서)** / 출처 / **최종 확인일** / 상태(운영중·종료·휴관)

### 4.5 인물 노드 규칙

- **서사 규칙**: 성취담이 아니라 **시행착오담** + 아이와 같은 나이 시절 일화 중심 (고생담 노출 집단이 성취담 집단보다 동기·성취가 높았다는 연구 근거: Lin-Siegler et al., 2016)
- **방향 규칙**: 책→현장이 아니라 **현장→인물** (체험처 카드에 인물 이야기 한 토막)
- **편성 규칙**: 반고정관념 쿼터에 인물 포함 (성별·신분 반전 스토리: 메리 애닝, 장영실 등). 기존 위인전집의 왕·장군·남성 편향 교정
- **선정 규칙**: 근현대 인물은 평가 논쟁 리스크 → 선정 기준 명문화, 평가가 안정된 과학·예술·탐험 인물 우선
- **금지**: '나의 롤모델 지정' 기능 (위인은 창문이지 과녁이 아니다)

### 4.6 도서 노드 규칙 (v0.4 신설)

- **구축**: ISBN 마스터(서지정보유통지원시스템) + KDC 분류 + 제목·소개글 임베딩(BGE-M3)으로 주제 태그 자동 매핑 → 사람 검수 큐. 별도 태스크가 아니라 **T1 확장으로 흡수**
- **노출**: 주제관 '더 깊이' 칸 — 연령 밴드별 3~5권, **소장 도서관·대출 가능 여부 표시**(정보나루). 도서관 대출 우선, 구매 링크 후순위 (중립성·헌장 3조)
- **체험 연결**: 체험 카드에 "가기 전에 볼 책 / 다녀와서 볼 책" — 촉발→유지의 다리. 타이밍 원칙: 현장→책 (현충사 다녀온 날 밤이 이순신 책을 건넬 순간)
- **신선도의 도서판**: 대출 가능 여부를 함께 제공 — 빌리러 가서 허탕 = 만료 정보 노출과 동급의 품질 사고
- **추천 API 사용 규칙**: recommandList('함께 빌린 책')는 그래프 엣지·후보 풀 전용, 화면 추천 순위로 직결 금지 (§4.2). 사서추천 API는 검증 품질 풀로 사용
- **금지**: 필독·권장·수준별 등 독서교육 언어 (추천 문구는 "이 세계를 더 볼 수 있는 책"까지만 — 흥미가 숙제가 되는 순간 G4 위반), 독서 레벨 지수 노출 (새로운 등급화)
- **스코프**: 도서는 지도의 부속. 독서기록·서평 서비스로 확장하지 않는다

## 6. 수집·운영 파이프라인 (제품의 본체)

- **Tier 1**: 공공 오픈API 정기 수집 (일·주 단위)
- **Tier 2**: 기관 홈페이지 크롤링 + 변경 감지 (국공립 박물관·과학관·도서관 공지)
- **Tier 3**: 제휴 콘솔 — 기관·지자체 직접 등록/갱신 (신선도 책임을 공급자에게 이전하는 장기 설계)
- **Tier 4**: 사용자 검증 — "다녀왔어요" 체크·오류 제보 = 신선도 크라우드소싱 + 실사용 정보(체감 연령·혼잡·주차)
- **자동 분류**: 수집 텍스트에 주제 태그 자동 부여(텍스트 분류) + 사람 검수 큐. 태그 사전은 관리형 통제(스프롤 방지)
- **신선도 규칙**: 출처별 갱신 주기 정의 / 기간 만료 자동 숨김 / 최종 확인일 노출 / 제보 즉시 반영 / **"만료 정보 노출 0건" = 품질 제1지표** (도서는 '대출 가능 여부'가 이에 해당)

## 7. 관심사 사전 구축 (착수 과제 #1)

**방향 확정**: 수요(어린이 쪽) 기점. 진로 데이터는 대문 재료가 아니라 **연결층 + 사각지대(먼 세계) 편성 재료**.
근거: ① 진로 데이터는 공급 측 행정 언어(크레존 실패 원인의 재연 위험) ② 커버리지 갭(슬라임·포크레인·마인크래프트 등은 직업 기점에서 미등장) ③ 진로 데이터에는 인기·연령·계절성 분포 정보가 0.

**파이프라인 (4단계)**
1. 수요 코퍼스 수집 (§8-A) → 명사구 추출
2. BGE-M3 클러스터링 + 동의어 통합 → 관심사 후보 + **연령대별 빈도 프로파일** (미취학 / 초저 / 초중·고 밴드 분리 — 공룡은 4~7세 피크, 아이돌은 초등 고학년)
3. 관심사 × {체험처·인물·직업·도서} 연결 스코어링 (MAPS FAISS + 행동 엣지 RRF 융합)
4. 2×2 배치:

| | 연결 高 | 연결 低 |
|---|---|---|
| **수요 高** | 주제관 1차 편성 | 대문 유지 + 다리 설계 과제 |
| **수요 低** | '먼 세계' 편성 재료 (반고정관념 쿼터 포함) | 보류 |

**주의**: 검색·구매·대출은 부모(사서) 매개 편향 → 단일 소스 확정 금지, 삼각측량 / 유튜브류는 자극≠흥미(G5) 오염 → 보조 지표로만

## 부록. 토론 이력 요약 (4라운드)

1. **1차 (어린이 진로 일반)**: 목적함수를 "진로 결정 정확도"에서 "열린 선택지 수 × 자기이해 데이터 질"로 재정의. 검사·매칭·등급 삭제, 경험층·관찰층·이력층 3층 합의 (산출물 7건 별도)
2. **2차 (흥미 서비스) + 수정 라운드**: "흥미는 찾는 게 아니라 기르는 것"(G1~G5) 도출 → 의뢰인 의도 재명제화로 교정 — 웹은 활동 공간이 아니라 정보 계층, 실체는 **분야별 체험활동 지도의 상시 수집·구조화·제공**(H1~H5)
3. **3차 (구조 확정)**: 학과·직업의 3층 부착(Layer A/B/C) → 인물 노드 추가(직업 계보 one-to-many) → 대문 우선순위 확정(관심사 > 인물 > 하는일) → 관심사 사전은 수요 기점, 진로 데이터는 연결층·사각지대 재료
4. **4차 (도서 축)**: 책을 **상시 재고**로 승격 — 부패 없음(H2 면제)·전국 균등(지역 격차 해법)·재방문 최저비용(G3)·한계비용≈0. 기성 API 조합으로 구축 난이도 하락: recommandList '함께 빌린 책'은 그래프 **행동 엣지**로(화면 순위 직결 금지), 사서추천은 검증 품질 풀로. 금지선: 독서교육 언어·레벨 지수 노출
