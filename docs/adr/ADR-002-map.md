# ADR-002 — 지도: Leaflet + OpenStreetMap

- 상태: 채택 · 2026-07-04

## 맥락
'지도 둘러보기' 화면과 각 체험 카드의 길찾기(현실 전환 CTA)에 지도가 필요하다. 무인 실행이라 API 키 발급·과금 승인 절차가 파이프라인을 끊으면 안 된다.

## 결정
**Leaflet** + **OpenStreetMap 타일**(`tile.openstreetmap.org`). 길찾기 링크는 OSM 좌표 딥링크(`openstreetmap.org/?mlat=&mlon=`).

## 근거
- 키 불필요 → 무인성 유지(네이버/카카오/구글 지도는 키·도메인 등록 필요).
- 좌표만 있으면 마커·팝업·딥링크 구성 가능. 수집 데이터가 이미 lat/lng 보유.
- 헌장 4조(화면은 예고편, 현실 전환)에 부합: 1차 CTA=길찾기.

## 대안
네이버/카카오 지도 API — 키·상업이용 승인 필요로 무인 원칙 위배. 반려.

## 결과
CDN Leaflet(오프라인 시 타일 미표시 가능 → 마커 좌표·길찾기 링크는 항상 동작). 지도 API는 `운영중`만 표시(신선도).
