# ADR-005 — Phase 0.5 관심사 추출 정식 편입 (운영 핵심 수정)

- 상태: 채택 · 2026-07-04

## 맥락
MVP는 기획서 §7(관심사 사전 구축)을 생략하고 주제 5개를 `common.py`에 하드코딩했다. 정보나루·공공데이터 API를 받은 원래 목적("무엇이 실제로 인기 있나 찾기")을 쓰지 못했다.

## 결정
운영 트랙에 **Phase 0.5**를 Phase 0과 1 사이에 정식 편입.
- 수요 = **정보나루 인기대출 실데이터**(연령대별 4파일, 4000행, 실제 대출건수).
- 연결 = **TourAPI 문화시설 실데이터**(2680건) 명칭 매칭.
- 관리형 후보 사전(스프롤 방지, 기획서 §6)에 실데이터를 붙여 2×2(수요×연결) → 상위 12개 자동 채택.
- `common.load_topics()`는 하드코딩 딕셔너리를 두지 않고, 산출물 없으면 **명시적 에러**(조용한 하드코딩 대체 차단, §2.6).

## 결과
- 채택 12: 그림·역사와위인·과학실험·동물·몸과건강·우주·식물·바다생물·운동·요리·곤충·음악.
- **MVP의 공룡·로봇이 실데이터 순위 밖으로 탈락** — 하드코딩 ≠ 실수요를 실증.
- MVP 5개는 `data/topics/mvp_reference.json`에 교차검증용 보존.

## 근거
API의 원래 취득 목적 복원. 주제가 트렌드에 따라 재추출(분기 배치) 가능(§8).
